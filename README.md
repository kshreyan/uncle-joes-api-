# uncle-joes-api-
Uncle Joes API

Team Members: Mihir Menon, Pranav Reddy, Sean Prue, Kumar Shreyansh

The API for this project will serve as a connection between the BigQuery database and the frontend application. It will allow customers to browse meny items, view store locations, log in as Coffee Club members, access their order history, and check their loyalty points. It will retreive and combine data from multiple tables and also handle simple requests like calculating loyalty points based on past purchases, making the data useable, and meaningful for the frontend.
